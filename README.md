# Dress2ndChance
Plateforme e-commerce de vêtements de seconde main avec recherche visuelle par image
## Description
Dress2ndChanc est une plateforme moderne dédiée à l'achat et la vente de vêtements d'occasion. Développée avec une approche UX centrée utilisateur, elle offre une expérience fluide et intuitive pour connecter acheteurs et vendeurs.
## Fonctionnalités
## Recherche & Navigation

Recherche par image - Upload photo ou webcam pour trouver des articles similaires
Filtres avancés - Prix, taille, distance, marque, date de dépôt
Catégories - Femmes, Hommes, Enfants, Accessoires, Promotions
Recherche textuelle - Avec suggestions et auto-complétion

## Espace Acheteur

Système de favoris
Panier avec récapitulatif détaillé
Paiement sécurisé (CB, PayPal)
Calcul automatique des frais de livraison
Suivi des commandes

## Espace Vendeur

Dépôt d'articles simplifié
4 types de profils : Particulier, Professionnel, Occasionnel, Boutique
Gestion des annonces (modification, suppression)
Application de promotions
Tableau de bord vendeur

## Gestion de Compte

Authentification sécurisée
Profil utilisateur personnalisable
Gestion des informations bancaires (IBAN)
Historique d'achats et de ventes
Paramètres de confidentialité

## Communication

Messagerie intégrée acheteur-vendeur
Notifications en temps réel
Support client multi-canal (mail, téléphone, chat)
FAQ dynamique

## Points Forts UX

Navigation intuitive - Architecture d'information claire
Icônes dynamiques - Changement visuel selon l'état de connexion
Filtres rapides - Accessibles en un clic depuis la page d'accueil
Transparence - Récapitulatifs détaillés à chaque étape
Responsive design - Optimisé mobile, tablette et desktop

## Innovation Clé
Recherche Visuelle
La recherche par image est LA fonctionnalité différenciante, développée suite aux retours utilisateurs :

"Les utilisateurs ont souvent une image précise en tête mais éprouvent des difficultés à décrire précisément le produit recherché."

Cette innovation offre :

Méthode rapide et intuitive
Résultats plus pertinents
Avantage compétitif face aux plateformes concurrentes

## Exemple d'Usage
Tâche : "Trouver tous les sweat-shirts mis en vente depuis hier dans une taille M de la marque Nike"
Solution :

Recherche textuelle "sweat-shirt"
Filtres : Taille → M, Marque → Nike, Date → Dernières 24h
Affichage des résultats en temps réel

## Méthodologie
Ce prototype a été développé selon une démarche UX complète :

Recueil des besoins - Interviews utilisateurs
Analyse de tâches - Identification des parcours critiques
Étude de l'existant - Benchmark des plateformes concurrentes
Prototypage - Maquettes moyenne fidélité
Tests utilisateurs - Itérations basées sur les retours


## Technologies Envisagées

Frontend : React.js / Vue.js
Backend : Node.js / Django
Base de données : PostgreSQL
Recherche visuelle : TensorFlow / AWS Rekognition
Paiement : Stripe API
Messagerie : WebSocket / Socket.io
